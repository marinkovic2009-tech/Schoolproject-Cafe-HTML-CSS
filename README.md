This repository contains a school project created to demonstrate fundamental skills in **HTML** and **CSS**.  
The focus is on understanding the structure and styling of web pages without any advanced features, animations, or interactivity.

---

## Overview

- **Objective:** Learn and practice the basics of web development.  
- **Scope:** 5 HTML files and 1 CSS file.  
- **Audience:** Beginners and anyone interested in understanding basic HTML and CSS structure.

---

## Included Files
| File             | Description                                      |
|-----------------|--------------------------------------------------|
| `index.html`     | Main page linking to all other sections/pages   |
| `about.html`     | "About Us" page with information about the cafe |
| `menu.html`      | Page showing the menu or offerings              |
| `reservation.html` | Page with a reservation form or booking example |
| `contact.html`   | Contact page with basic contact information    |
| `styles.css`     | CSS file styling all HTML pages                 |

---

## How to Use

1. **Clone or download** this repository.  
2. **Open** any `.html` file in a web browser.  
3. The **CSS styles** will automatically be applied.  

_No installation or setup is required._

---

## Learning Outcomes

By working with this project, you will gain practical knowledge in:

- Basic HTML structure (head, body, headings, paragraphs, links, lists)  
- Linking a CSS stylesheet to HTML  
- Applying basic styling: colors, fonts, spacing, and layout  

This project is intended solely for educational purposes and to strengthen fundamental web development skills.
